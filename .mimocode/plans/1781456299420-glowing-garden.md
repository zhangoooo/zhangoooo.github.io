# Plan: Create home.html Personal Blog Homepage

## Goal
Create a standalone `home.html` file — a modern dark-theme personal blog homepage with all major sections.

## File to Create
- `/mnt/e/Code/web/zhangoooo.github.io/home.html` — single self-contained HTML file with inline CSS and JS

## Design Approach
- **Dark theme**: Deep background (#0a0a0a / #111) + accent color (cyan/blue) + light text
- **Fully self-contained**: All CSS and JS inline (no external files needed)
- **Responsive**: Mobile-first with hamburger menu
- **No external dependencies except fonts**: Use Google Fonts (Inter) via CDN, Ionicons via CDN for icons
- **Leverage existing images**: Reference assets from `image/` directory where appropriate

## Page Sections

### 1. Header / Navigation
- Fixed top navbar with site name/logo on left
- Navigation links: Home, Articles, Categories, About, Contact
- Dark translucent background with blur effect
- Mobile: hamburger menu toggle

### 2. Hero Section
- Large greeting with animated typing effect or simple fade-in
- Profile avatar (use existing `image/home.png` or fallback)
- Name + short bio/tagline
- Social media icon links (GitHub, Twitter/X, Email)
- CTA button to latest articles

### 3. Latest Posts
- Section title: "最新文章"
- Grid of 3–4 post cards (responsive: 3 cols desktop, 2 tablet, 1 mobile)
- Each card: thumbnail image, date, category tag, title, short excerpt, "Read more" link
- Use placeholder content (Chinese)

### 4. Featured Posts
- Section title: "精选文章"
- Horizontal scroll or 2-column layout for featured/highlighted posts
- Larger cards with image background overlay
- Use existing images from `image/` (featured-1..5.png)

### 5. Categories / Tags
- Section title: "分类"
- Visual grid of category cards with icons and post count
- Categories: 技术笔记, 前端开发, 生活随笔, 学习记录, 项目实践
- Each card has a subtle gradient background

### 6. About Me
- Section title: "关于我"
- Two-column layout: text left, image/decoration right
- Brief personal intro paragraph
- Skills/tech stack tags
- Social links with hover effects

### 7. Footer
- Site name + copyright
- Quick navigation links
- Social media icons
- "Back to top" button

## Technical Details

### CSS Architecture (inline `<style>`)
- CSS custom properties for colors, spacing, transitions
- Dark color palette:
  - `--bg-primary: #0a0a0a`
  - `--bg-secondary: #141414`
  - `--bg-card: #1a1a2e`
  - `--accent: #00d4ff` (cyan)
  - `--accent-2: #7b61ff` (purple)
  - `--text-primary: #ffffff`
  - `--text-secondary: #a0a0b0`
- Smooth scroll, hover animations, card hover lift effects
- CSS Grid + Flexbox for layout

### JavaScript (inline `<script>`)
- Mobile nav toggle
- Smooth scroll for anchor links
- Intersection Observer for scroll-reveal animations
- "Back to top" button visibility toggle
